* Carry Bonus Factor (C)

No scores should be permanently stored.

Scores must always be calculated dynamically using the current machine settings.

# SCORING ENGINE

Replace the previous scoring formula entirely.

Use the following calculation model:

Production Score:

P = (Actual ÷ Target) × PC

Quality Score:

Q = QC − ((Defect Percentage ÷ 100) × K)

Base Score:

Base = (P × wp) + (Q × wq)

Carry Bonus:

Calculate the next shift production ratio:

NSR = Next Shift Actual ÷ Next Shift Target

Carry Bonus = max(0, (NSR − 1) × C)

Final Score:

Final Score = Base + Carry Bonus

The scoring engine must work dynamically for every production record.
